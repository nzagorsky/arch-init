# arch-init
Installs barebone Arch (both for EFI and legacy).
  
**Shell**: zsh  
**Editor**: neovim  
**Network**: NetworkManager  
**Display**: Xorg  
**Other**: Docker, tmux  
  
# Usage: 

Boot live CD, partition the disk and connect to the network.

    curl -sfL git.io/JB9ki > run.sh
    bash run.sh
